## Utilisation

- **HTML** : Ouvrir dans un navigateur. Ctrl+P pour exporter en PDF.
- **CSV** : Ouvrir dans Excel (séparateur point-virgule pour le français).
- **Python** : Exécuter `python generateur_package.py` pour régénérer (nécessite Python).
