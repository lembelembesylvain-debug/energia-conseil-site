VIII. Dimensionnements techniques

  • Chaudière : Hargassner NanoPK12 — 12 kW — classe A+ ; Flamme Verte 7 étoiles ; rendement ETAS 81 % ; silo granulés 5 T avec extracteur automatique.
  • VMC : simple flux hygroréglable (MS Plomberie).
  • ITI : ISONAT 145 mm, λ = 0,038 W/m·K, R = 4,05 m²·K/W (calcul : R = 0,145/0,038).
  • Combles : 2 couches ISONAT (180 + 145 mm) — R = 9,05 m²·K/W (somme des R des couches).
  • Planchers : chape béton cellulaire 180 mm — R ≥ 3,0 m²·K/W.
  • Menuiseries : PVC triple vitrage Uw ≤ 1,1 W/m²·K — gamme ELITH.
