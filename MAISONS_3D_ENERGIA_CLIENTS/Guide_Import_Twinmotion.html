Guide d’import des maisons 3D dans Twinmotion

Projet : MAISONS_3D_ENERGIA_CLIENTS
Logiciel : Twinmotion 2025.2
Format : OBJ + MTL (Wavefront)

1. Prérequis

  • Twinmotion 2025.2 (ou version compatible) installé
  • Fichiers .obj et .mtl dans le même dossier
  • Unités du modèle : mètres (1:1)

2. Import du fichier OBJ

Étape 1 : Ouvrir Twinmotion
Étape 2 : Menu File > Import (ou raccourci Ctrl+I)
Étape 3 : Sélectionner le fichier maison_[client].obj
Étape 4 : Cliquer sur Ouvrir

Le modèle s’affiche dans la scène, centré à l’origine (0, 0, 0).

3. Vérifications après import

  • Échelle : 1 unité = 1 mètre (pas de mise à l’échelle nécessaire)
  • Position : La base de la maison est au sol (Z = 0)
  • Matériaux : Les matériaux MTL (murs, toit, sol) sont chargés automatiquement

4. Structure des dossiers

MAISONS_3D_ENERGIA_CLIENTS/
├── M_RICHARD_UNIAS/      → maison_richard.obj + .mtl
├── MME_FONTANAY_MONTROND/
├── M_ROYER_CHALAIN/
├── M_SEUX_RICAMARIE/
├── M_GUION/
└── M_YANG_COLLONGES/

5. Problèmes courants

Matériaux non affichés : Vérifier que le fichier .mtl est dans le même dossier que le .obj et porte le même nom de base.

Modèle trop petit ou trop grand : Les dimensions sont en mètres. Vérifier les paramètres d’unité de la scène Twinmotion (mètres par défaut).

Modèle inversé ou mal orienté : Utiliser les outils de transformation (rotation, miroir) dans Twinmotion si nécessaire.

6. Étapes suivantes

Après l’import, consulter le Guide_Finalisation_Rendu.html pour :

  • appliquer des textures réalistes
  • configurer l’éclairage
  • créer les rendus « Avant / Après » travaux
  • exporter des images 4K et des visites 360°
