Guide de finalisation et rendu pour présentations commerciales

Objectif : Obtenir des visuels réalistes « Avant / Après » pour les clients ENERGIA-CONSEIL IA®
Logiciel : Twinmotion 2025.2

1. Appliquer les textures réalistes

1.1 Bibliothèque Twinmotion

  • Ouvrir le panneau Library (bibliothèque)
  • Catégorie Materials > sous-catégories : Bricks, Concrete, Stone, Wood, etc.

1.2 Correspondance matériaux par type de maison

Type maisonMursToitMenuiseries
Pierre / moellons (Richard, Royer)Stone > Pierre locale, moellonsRoofing > Tuiles mécaniquesWood > Bois naturel ou PVC blanc
Enduit (Fontanay)Concrete > Enduit blanc/crèmeRoofing > TuilesWood / PVC
Blocs béton (Seux)Concrete > Béton gris, parpaingRoofing > TuilesPVC ou aluminium
Pierre calcaire (Yang)Stone > Pierre calcaire doréeRoofing > Tuiles ou ardoiseBois / aluminium

1.3 Application

Sélectionner l’objet (maison) > Clic droit sur une face > Replace material > Choisir une texture de la bibliothèque

2. Éclairage et environnement

  • Soleil : Ajuster l’heure (ex. 14h) et l’orientation pour mettre en valeur les façades
  • Ciel : Ciel dégagé ou légèrement nuageux
  • Sol : Ajouter un terrain ou une base paysagée (gazon, allée)
  • Environnement : Arbres, haies, mobilier léger pour le contexte

3. Créer les versions « Avant » et « Après » travaux

3.1 Version « Avant »

  • Conserver le modèle importé tel quel
  • Textures correspondant à l’état actuel (murs fatigués, toiture ancienne)
  • Nommer la scène ou l’objet : Maison_[Client]_Avant

3.2 Version « Après »

  1. Dupliquer la scène ou l’objet
  2. Modifications à appliquer :
    • ITE (isolation thermique extérieure) : Augmenter légèrement l’épaisseur des murs visuellement, ou appliquer une texture « enduit neuf » plus propre
    • Toiture : Changer la texture (tuiles neuves, ardoise, etc.)
    • Menuiseries : Nouvelles fenêtres (PVC blanc, bois clair)
    • Volets / stores : Si prévus au devis
  3. Nommer : Maison_[Client]_Apres

4. Rendu 4K et visite 360°

4.1 Image fixe 4K

Menu Presentation > Image > Résolution 3840 x 2160 (4K) > Choisir le point de vue > Render

4.2 Visite 360°

Menu Presentation > 360° Panorama > Positionner la caméra au centre de la visite > Render

5. Export et livrables

  • Images : PNG ou JPG haute résolution (4K recommandé)
  • 360° : Format adapté à la diffusion web (Twinmotion Cloud ou export HTML)
  • Noms de fichiers : Maison_Richard_Avant.png, Maison_Richard_Apres.png, etc.
Conseil : Pour un rendu homogène entre clients, utiliser les mêmes paramètres de soleil, ciel et environnement pour toutes les maisons.
